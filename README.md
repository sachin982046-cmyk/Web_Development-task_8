# Simple Blog Layout Using Bootstrap 5
  Objective

Build a responsive blog webpage using Bootstrap 5 CDN that includes:

A navigation bar

Blog post cards

Responsive grid layout

Footer with social icons

# Tools & Technologies

HTML5

CSS3

Bootstrap 5 (CDN)

VS Code

Any Web Browser

# Project Description

This project demonstrates how to build a simple and responsive blog layout using Bootstrap 5.
The site includes a navbar, three blog cards, and a footer.
Bootstrap grid system and utility classes are used to ensure responsiveness on all screen sizes.

# Features

✔ Fully responsive design
✔ Bootstrap 5 navbar
✔ Blog post cards with image, title, description, and button
✔ Footer with social icons
✔ Clean layout using Bootstrap utility classes

# How to Run the Project

Download or clone this repository.

Open the project folder in VS Code.

Open the file index.html in your browser.

Customize images, text, and styles if needed.

# Steps to Build the Blog Page

Add Bootstrap 5 CSS and JS CDN links

Create navbar using navbar classes

Add cards inside a container → row → col-md-4 structure

Insert footer with social icons using Bootstrap Icons CDN

Test responsiveness using Chrome DevTools
